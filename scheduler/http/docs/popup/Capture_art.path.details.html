TV Scheduler Pro


Capture Details Art Path
This option allows TV Scheduler Pro to write to an NFO file a thumb tag containing image location details, this info can be used to populate 3rd party TV viewing and playback tool (especially Kodi) with info about what TV Scheduler Pro has captured.
Note: The tag takes the name from the capture epg title and strips out spaces and special characters and adds .jpg as the extension
e.g. an epg title of "American Dad!: Special Episodes" would tag the art as "<thumb>Path to Folder\AmericanDadSpecialEpisodes.jpg</thumb>"

Options available:

  • none = No thumb tag written
  • same = Thumb tag is written to point to the same location as the captured file
  • path = a path specifying where you want the thumb tags to point. Allows a common images folder
