TV Scheduler Pro


Capture Details Episode NFO Path
This option allows TV Scheduler Pro to write an NFO file containing capture details, this info can be used to populate 3rd party TV viewing and playback tool (especially Kodi) with info about what TV Scheduler Pro has captured.

Options available:

  • none = No nfo details are written
  • same = Details file is written to the same location as the captured file. Can be captures all in one folder or separate folders.
  • path = a path specifying where you want the details file to be written. Included for debugging or postprocessing etc.
