MatchList SPAN condition
The Span conditions have two types.

Type 0 (Start) is a time span that a program must start within.

Type 1 (Duration) is a duration the program must be within.

A span condition can help narrow down the number of matches to a Match List, eg.
(Type 0) match all news programs between 5:30 and 7:30, or
(Type 1) match all Soccer games between 1 and 3 hours long.
