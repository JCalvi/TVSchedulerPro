MatchList TEXT matches
This type of match allows you to match a text string or RegEx query with any of the text fields of an Epg item.

For normal none RegEx matches you can use a wild card * at the beginning or end or both the beginning and end to show the following:

*TEXT - match ends with
TEXT* - match starts with
*TEXT* - match contains
TEXT - must match exactly

You can also set a Case insensitive match that will match TexT Text or TEXT etc.

RegEx Note:
RegEx is an advanced text query language called Regular Expressions.

Regular Expressions can sometimes be hard to understand and write but for simple matches they are relatively easy to create, for example:

".*News.*" :- this would match all program names with the word "News" in them.

"(?i).*news.*" :- this would match all programs with the word "news" in them using Case-insensitive matching, i.e. News or nEws or NEWS or news etc

".*[Vv]oyager" :- this would match a program name that ended with Voyager or voyager.

You can also use the Text, Subtitle Or Description Matcher with or without the single line modifier (?s) to match complicated sequences eg.

Two match conditions:
"(?i).*AFL.*" :- Title Match &...
"(?i).*(Richmond|Tigers).*" :- Title, Subtitle or Description :- this would match AFL in the title and Richmond in the subtitle. In this case AFL must be in the title, but Richmond or Tigers can be in either subtitle or description.

Single Match Conditions:
"(?is).*AFL.*(Richmond|Tigers).*" :- this would match all programs with AFL and Richmond or Tigers (Case Insensitive) within all the words added together in the title, subtitle and description in that order. This would match AFL in the title and Richmond in the subtitle as above, but could also match AFL in the subtitle and Tigers in the description.

"(?i).*AFL.*(Richmond|Tigers).*" :- omitting the "s" modifier means the match must be ALL contained within one of the fields, not spread over them. This would not match AFL in the title and Richmond in the subtitle, but would match AFL...Tigers in the description.
