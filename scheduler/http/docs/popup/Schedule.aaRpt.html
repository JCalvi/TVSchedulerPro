Add Repeat=False to Auto-Add Matchlists
This adds Repeat = False as a match requirement to Auto-Add matchlist items created via the epg.

The default (false) is to not add the repeat match, i.e match all items regardless of repeat.

e.g.
If this setting is "True" and we make a new auto-add for "Media Watch" and the show is on at 9:30pm on Monday and a repeat is on 11am on Tuesday and the guide (epg) data correctly has the second one on Tuesday with the repeat flag set then TVSP will match the first (Monday) and record it but exclude the second (Tuesday) repeat.
