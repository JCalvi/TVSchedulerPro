Email Server options
To Address: The email address you want the notification sent to, this can be multiple addresses separated by commas.

From Address: This should be your email address or the email address you want the notification emails to look like they are coming from.

Server Address: The address of your SMTP email server.

Server Port: The port your server uses, if you are using security like SSL or STARTTLS you may need to set this to your servers expected port number.

Use Authentication: Check this to use the user name and password to authenticate your connection.

Security: Some SMTP servers require security, this option will use the selected connection security, check your server requirements and capabilities for available protocols, and choose the highest supported.
