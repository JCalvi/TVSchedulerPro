Created From Inexact Matching
Sets TVSP to apply inexact title matching when trying to establish where a schedule was "Created From".

Inexact Matching checks if the old title is a substring of the new one and vice versa, and also ignores case.
Exact Matching requires the two titles to be equal.

e.g.
Inexact: "House" will match "house: md"
Exact : "House" will only match "House"
