Device Selection $cardList



Currently Selected Devices:
These devices will be used by TV Scheduler Pro. You can change the order that TV Scheduler Pro will use them in by moving them up and down the list by clicking on the images. You can remove the device from the list by clicking the image next to the device you want to remove.

Devices Available But Not Selected:
These devices are available but TV Scheduler Pro will not use them. Click the green to add them to the selected devices list for TV Scheduler Pro to be able to use them.

Showing Device IDs:
TV Scheduler Pro uses your device system ID to track its usage and specify what device is to be used for captures. This ID is a string with all the information required to uniquely identify the device, for this reason it can be quite long. By default the device list does not show this ID. To show the IDs for your devices click the "Show IDs" link in the title of the Currently Selected Devices, this will then show you the IDs for the devices you have selected. To hide them again click the "Hide IDs" link.

Missing Devices:
Devices that have been added to the selected device list but are no longer available due to being removed from the system or hardware/driver failure will show up in the selected device list with icon next to the name of the device. This means the device is no longer available and you should either fix the offending hardware/driver or remove the device from the selected device list.
