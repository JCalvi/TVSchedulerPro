Browser Agent to Theme Mapping
To map a browser type to a particular theme you can set a browser agent string to theme mapping. This means that that browser will always use the mapped theme, this is useful for browsers on phones and portable devices.

You can specify the simple there or a theme you have created that works with your devices browser but the default there is still usable on your main PC or HTPC browser.

In the text box above you will see your current browsers Agent string, this string is how a browser identifies itself to a web server, as you can see it is long and has a lot of information in it. You can delete most of it and just have a sub string of the data i.e. Firefox or MSIE 7.0 etc, the matching to the browser string is done as a sub string match so any instance of the partial string is considered a match. The match is case sensitive so be careful if typing in the string manually that you get the case correct.

To add a mapping for a browser to easiest way to do it is connect to this page form the browser in question select the theme you want to use and then click the add button. This will add a theme to agent map for the browser you are connecting with to the theme mapping. To test it go back the settings page and click the Show Keyboard Interface link.
