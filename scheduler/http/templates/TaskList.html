Tasks
Tasks can be used to launch external application to accomplish actions like process capture files, closing applications or restart device drivers. You can set the default task that will be used as a post capture task for all Schedule items that are added using the drop down box above. You can also set a Pre-Capture task that will run before any capture is run, this is useful for closing other applications or resetting a problem driver before a capture is attempted.

*Please Note:
For Pre-Capture Tasks, Capture Start Failed Tasks and No Data Flowing Task you need to set them up as follows or you may see timeout problems when running your tasks.

Delay = 0
When not capturing = False or Unchecked
Maximum concurrent tasks >= 20
Don't run within X minutes of a scheduled action = 0
