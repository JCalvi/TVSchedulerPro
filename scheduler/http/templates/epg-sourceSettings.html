File Source
The file source just loads XMLTV data from any file in the specified directory that has an extension of .xml.
Note: a folder without drive and path will be relative from the Windows Programdata folder
eg. xmltv = C:\Programdata\xmltv, use a full path to point somewhere else.
eg. D:\Somefolder\where\i\like\to\keep\my\xmltv

HTTP Web Source
This option downloads data from the specified URL.

Proxy Settings
If you need to use a proxy to connect to the Web enter the proxy in the HTTP Proxy Settings option.

Schedule Data Load
You can schedule to download and import data every day at a set time, TV Scheduler Pro will wake the system up from suspend or hibernate if it needs to to do this reload action.

You can also have a pre load task take place before the data load action, this allows a task to do an EPG data load using an external tool and save the data to disk before the EPG data is then loaded into TV Scheduler Pro.

Warn when EPG items overlap
Some EPG sources can have overlapping EPG items, this is never a good thing and is this occurs you can be warned when you view the EPG page.

Remove any epg items before current time
This option removes any programmes from the epg that start before the time the xmltv is loaded. Use this to clip xmltv files that have old data in them.

Merge double episodes
This option merges concurrent EPG programs that have the same name so when they are added using the Auto-Add or EPG they are added as a single schedule and thus a single capture file.

Search URL
This is the URL that will be used for external searches from the program details page, it will launch a new window to do the search. In the URL the following will be replaced with details form the program:
$TITLE = program title
$SUB = program sub title
$CAT = program category
