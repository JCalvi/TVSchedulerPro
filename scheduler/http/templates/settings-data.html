Settings and Options Backup
Save Settings
From the following list select which settings you would like to save then click the "Save Now" button and you will be prompted to save the settings file. The settings file is a standard ZIP file containing all of your selected settings in the one file.












Load Settings
Click the "Browse" button and when prompted browse to the settings file you would like to load. From the following list select which settings you would like to load form the settings file. Then enter the security ID and click the "Load Now" button.
